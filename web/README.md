# sisyphus web

Minimal static landing page (Vite + vanilla TypeScript). Copy-paste entry commands for the JobSpy pipeline and fit calculator.

## Develop

```bash
cd web && npm ci && npm run dev
```

## Build

```bash
cd web && npm ci && npm run build
```

Output: `web/dist/` (gitignored). Serve with any static file server, for example `npx serve dist` from the `web/` directory.
